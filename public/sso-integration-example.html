🚀 Chat Taiksu SSO

Exemplo de Integração com Autenticação Centralizada

📖 O que é SSO?

SSO (Single Sign-On) permite que sua aplicação use o sistema de autenticação centralizado do Taiksu. Não é necessário manter sessões locais - o Chat Taiksu valida tudo com o servidor central.

Benefícios:

  • Uma única autenticação para múltiplas aplicações
  • Dados do usuário sincronizados automaticamente
  • Sem necessidade de gerenciar senhas localmente
  • Suporte a múltiplos microsserviços

🔄 Fluxo de Autenticação

1️⃣ Sua aplicação obtém um token

Seu servidor faz login no SSO centralizado e recebe um token JWT

POST https://login.taiksu.com.br/api/login
Body: { email, password }
Response: { token: "eyJhbGc..." }

2️⃣ Token é enviado ao Chat Taiksu

Sua aplicação passa o token para validação

POST http://chat.taiksu.local:3000/api/auth/sso/validate
Body: { token: "eyJhbGc..." }
Response: { success: true, user: {...} }

3️⃣ Chat Taiksu valida com o SSO

Chat consulta o servidor SSO para confirmar autenticidade do token

GET https://login.taiksu.com.br/api/user/me
Headers: { Authorization: "Bearer eyJhbGc..." }
Response: { id, name, email, foto, ... }

4️⃣ Usuário sincronizado e sessão criada

Chat Taiksu cria/atualiza usuário localmente e estabelece sessão

Sessão criada com dados do usuário
Pode acessar: /chat/rooms, /dashboard, etc
Cookie: connect.sid={session_id}

💻 Como Implementar

Frontend (JavaScript)

// 1. Obter token (da sua aplicação SSO)
const token = localStorage.getItem('sso_token');

// 2. Validar no Chat Taiksu
fetch('http://chat.taiksu.local:3000/api/auth/sso/validate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ token })
})
.then(r => r.json())
.then(data => {
  if (data.success) {
    console.log('Autenticado:', data.user);
    // Redirecionar para chat
    window.location.href = 'http://chat.taiksu.local:3000/chat/rooms';
  }
});

Backend (Node.js)

// 1. Receber token do cliente
app.post('/auth/sso/validate', async (req, res) => {
  const { token } = req.body;

  // 2. Enviar para Chat Taiksu
  const response = await fetch('http://chat.taiksu.local:3000/api/auth/sso/validate', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ token })
  });

  const data = await response.json();
  res.json(data);
});

Variáveis de Ambiente

# .env
SSO_URL=https://login.taiksu.com.br
SSO_VALIDATE_ENDPOINT=/api/user/me
SSO_TIMEOUT=5000
ENABLE_SSO=true

📚 Endpoints Disponíveis

Validar Token

POST /api/auth/sso/validate
Content-Type: application/json

{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

Obter Usuário Autenticado

GET /api/auth/sso/me
Authorization: Bearer {token}
Accept: application/json

Fazer Logout

POST /api/auth/sso/logout
Authorization: Bearer {token}
