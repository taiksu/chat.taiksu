🎨 Widget Chat Taiksu - Exemplo de Integração

Bem-vindo! Este é um exemplo de como integrar o Chat Taiksu widget em sua aplicação cliente. Abra o console do navegador ou clique no botão de chat no canto inferior direito da tela.

📋 Requisitos

  • Servidor Chat Taiksu rodando
  • ID da sala de chat válido
  • Acesso à API do servidor

🚀 Como Integrar

Adicione o script do widget na sua página HTML:

<!-- Incluir script do widget -->
<script src="http://localhost:3000/js/taiksu-widget.js"></script>

<!-- Inicializar widget -->
<script>
  TaiksuChat.init({
    serverUrl: 'http://localhost:3000',
    roomId: 'id-da-sala-aqui',
    title: 'Suporte ao Cliente',
    position: 'bottom-right'
  });
</script>

⚙️ Opções de Configuração

Opção Tipo Descrição
serverUrl String URL do servidor Chat Taiksu
roomId String ID da sala de chat
title String Título do widget
position String Posição: bottom-right, bottom-left, etc
autoOpen Boolean Abrir automaticamente (padrão: false)

🔑 Obter ID da Sala

Para encontrar o ID da sua sala de chat:

  1. Faça login no Chat Taiksu
  2. Acesse a sala desejada
  3. O ID da sala aparece na URL: /chat/room/{roomId}

💡 Exemplos

Exemplo 1: Configuração Básica

TaiksuChat.init({
  serverUrl: 'http://localhost:3000',
  roomId: 'abc-123-def'
});

Exemplo 2: Configuração Completa

TaiksuChat.init({
  serverUrl: 'https://chat.seudomain.com',
  roomId: 'sala-vendas-2026',
  title: 'Suporte de Vendas',
  position: 'bottom-right',
  autoOpen: false
});

Exemplo 3: Múltiplos Widgets

// Widget 1 - Suporte
TaiksuChat.init({
  serverUrl: 'http://localhost:3000',
  roomId: 'sala-suporte',
  title: 'Suporte'
});

// Widget 2 - Vendas (se implementado para múltiplos)
// TaiksuChat2.init({...});
📝 Nota: O widget atual suporta uma única instância. Para múltiplos widgets, será necessário estender o código.

🛠️ Troubleshooting

Widget não aparece

Verifique:

  • Se o servidor está rodando na URL configurada
  • Se o roomId é válido
  • Se não há erros no console do navegador

CORS Error

Se receber erro de CORS:

  • Verifique se o servidor tem CORS configurado
  • Adicione seu domínio à lista de origens permitidas
