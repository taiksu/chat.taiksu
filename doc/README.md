# Documentacao Chat Taiksu

Esta pasta foi reorganizada para servir como base oficial de consulta do projeto.

## Documentos

1. `ARQUITETURA.md`: estrutura da aplicacao, camadas e componentes.
2. `ROTAS_E_APIS.md`: mapa completo de rotas web e APIs.
3. `FLUXOS_DE_USO.md`: fluxos reais de autenticacao, chat e operacao.
4. `BANCO_DE_DADOS.md`: modelos, tabelas e relacoes.
5. `OPERACAO_E_MANUTENCAO.md`: variaveis de ambiente, deploy, runbook e evolucao.

## Ordem recomendada para onboarding

1. Ler `ARQUITETURA.md`
2. Ler `FLUXOS_DE_USO.md`
3. Consultar `ROTAS_E_APIS.md`
4. Consultar `BANCO_DE_DADOS.md`
5. Usar `OPERACAO_E_MANUTENCAO.md` no dia a dia

## Escopo desta documentacao

- Baseada no codigo atual em `src/`
- Focada em comportamento em execucao
- Escrita para manutencao futura e atualizacoes seguras
